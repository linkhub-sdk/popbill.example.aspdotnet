popbill.example.aspdotnet
======================

팝빌 SDK Example for ASP.NET

+ SDK 개발환경은 .NET Framework V3.x 이상 또는 v2.x 기반에서 제작
+ API 호출과 응답에 사용되는 메시지 인코딩 방식은 “UTF-8 Character-Set” 적용

1. 전자세금계산서
2. 전자명세서
3. 현금영수증
4. 전자세금계산서 홈택스 연동
5. 현금영수증 홈택스 연동
6. 휴폐업조회
7. 문자
8. 팩스
9. 카카오톡(알림톡/친구톡)

[ASP.NET SDK 적용방법](http://blog.linkhub.co.kr/4022/)
